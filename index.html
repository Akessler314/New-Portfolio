Change of Season TCG

Change of Seasons TCG

React Web Game (MERN Stack)

This application is a multiplayer online fantasy card game. Users create an account and battle one and other in real time using pre-made decks of fantasy cards. This application was made for my final project with a group in our Full Stack Development Course, where we had 3 weeks to complete the project, but we are still making improvements to make the game marketable.

Technologies Used

  • React Framework, implementing the useContext useState and useEffect hooks
  • Socket.io to connect 2 users into a match
  • Drag and Drop React Libary to move cards around on the board
  • Passport for logging in users and protecting routes
  • BcryptJs to protect user's passwords
  • MongoDB to store information about the user (username, userID ect...) as well as the card information to dynamically create them rather than make each individual one
  • Animate CSS for animations on the site
  • Custom scripting language to handle some of the playing cards logic (see client/src/components/GameBoard/cardScript.js in repository)
